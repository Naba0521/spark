# spark

Spark English Centre — bilingual (MN/EN) marketing website for Ulaanbaatar.

## Stack

- Next.js (App Router, static export)
- TypeScript, Tailwind CSS
- next-intl

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000/mn](http://localhost:3000/mn).

## Build

```bash
npm run build
```

Static output is generated in `out/`.

## Deploy

Push to GitHub and deploy on Vercel, or upload the `out/` folder to any static host.
